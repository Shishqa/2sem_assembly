## Лабораторная работа "printf"

Цель работы: написать аналог функции printf из языка C на языке ассемблера. Необходимо перенести минимальный, но достаточный набор флагов: `%s`, `%c`, `%d`, `%u`, `%b`, `%o`, `%x`, `%%`.
